Examples
  • sin(x), cos(x), tan(x)
  • x^2, (x+1)*(x-1)
  • x, x+1, 2*x (good for the new modes)
  • 1/x (discontinuity)
Modes: Cartesian plots y=f(x). Δ from x plots f(x)-x (so y=x becomes flat). Parallel number lines draws arrows mapping inputs to outputs.
Supported
Operators + - * / ^ %, parentheses, constants pi, e, parameter t (animatable), and functions sin, cos, tan, asin, acos, atan, sqrt, abs, ln, log, exp, floor, ceil, round, mod.
Tip: use * explicitly (e.g. 2*x).
